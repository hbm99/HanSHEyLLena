hanSHEyLLena tiene la posibilidad de recibir múltiples pipes. Se tiene un array donde cada posición representa una tubería y tiene un array de enteros de tamaño 2 donde se guardan los files descriptors correspondientes. Para esto se utiliza la función pipe() de C. Cada uno de los comandos indicados se guardará en el historial de comandos, con la excepción de cuando se escribe con ' ' delante de este, como se indica en la orientación.
